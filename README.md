Healthcare Appointment No-Show Prediction
Objective
Predict whether patients will miss appointments and identify factors
affecting attendance.
Tools
Python
Pandas
Scikit-learn
Matplotlib
Power BI
Workflow
Data cleaning
Feature engineering
Exploratory Data Analysis
Decision Tree model
Model evaluation
Dashboard creation
Key Insights
Around 20% of appointments were missed.
Teenagers had the highest no-show rate.
Seniors had the lowest no-show rate.
Longer waiting periods can influence attendance.
SMS reminders alone did not guarantee lower no-show rates.
Recommendations
Reduce waiting times.
Focus reminder strategies on high-risk patients.
Use predictive models for appointment scheduling.
